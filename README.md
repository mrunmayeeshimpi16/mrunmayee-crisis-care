# mrunmayee-crisis-care
Crisis Care App is an emergency response platform that provides quick access to ambulance, doctor consultation, pharmacy support, emergency food delivery, and transport assistance. It also includes an SOS alert system that shares user details like location, phone number, and username for immediate help during emergencies.
